# Project Operations Dashboard — Construction & Field Teams

Built in Power BI using construction project data modeled after 
large-scale energy project environments ($100M+ scope).

## What it tracks
- Budget vs Actual cost by project and category
- Cumulative spend trends over time
- Project schedule status (On Track / At Risk / Delayed)
- Risk register with RAG status
- Subcontractor cost variance by project

## Tools
Power BI | DAX | Excel | Star Schema Data Model

## Pages
1. Executive Summary
2. Cost Breakdown
3. Schedule & Risk
4. Contractor Costs
